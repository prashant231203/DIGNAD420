6 Time Series

• Dates:  
• dseries class:  

Dynare provides a Matlab/Octave class for handling time series data, which is based on a class for handling dates. Dynare also provides a new type for dates, so that the basic user do not have to worry about class and methods for dates. Below, you will first find the class and methods used for creating and dealing with dates and then the class used for using time series.
